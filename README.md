# Offline Messenger App

## Описание

Offline Messenger App - это Android-приложение, предназначенное для обмена сообщениями без доступа к интернету. Приложение использует технологию Wi-Fi Direct для создания P2P-соединений между устройствами и mDNS для обнаружения сервисов в сети.

## Основные возможности

- **Обмен сообщениями**: Отправка и получение текстовых сообщений между устройствами без интернета.
- **Wi-Fi Direct**: Автоматическое создание P2P-соединений между устройствами.
- **Service Discovery**: Обнаружение сервисов с использованием mDNS.
- **Поддержка нескольких участников**: Возможность подключения нескольких устройств к одному Group Owner.

## Структура проекта

### Основные компоненты

#### 1. `WiFiServiceDiscoveryActivity`
Этот класс отвечает за обнаружение и подключение к сервисам Wi-Fi Direct. Он использует mDNS для поиска доступных сервисов и управляет подключениями к другим устройствам.

#### 2. `WiFiDirectServicesList`
Этот класс представляет собой список доступных сервисов, найденных с помощью Wi-Fi Direct. Он отображает устройства и позволяет пользователю выбрать, к какому сервису подключиться.

#### 3. `WiFiChatFragment`
Фрагмент, отвечающий за отображение чата. Он позволяет пользователям отправлять и получать сообщения, а также управляет историей сообщений и интерфейсом чата.

#### 4. `WiFiDirectBroadcastReceiver`
BroadcastReceiver, который обрабатывает события Wi-Fi Direct, такие как изменения состояния соединения и обнаружение устройств.

#### 5. `GroupOwnerSocketHandler`
Этот класс отвечает за создание и управление серверным сокетом на устройстве Group Owner. Он принимает подключения от клиентов и обрабатывает сообщения.

#### 6. `ClientSocketHandler`
Класс, который управляет клиентским сокетом. Он подключается к серверу Group Owner и отправляет/принимает сообщения.

#### 7. `ChatManager`
Класс, который управляет логикой чата, включая отправку и получение сообщений, обновление интерфейса и хранение истории сообщений.

## Установка

1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/FedorVisa/p2p-anroid-chat
    ```

2. Откройте проект в Android Studio.

3. Синхронизируйте зависимости Gradle.

4. Запустите приложение на вашем устройстве или эмуляторе.

## Использование

1. Включите Wi-Fi Direct на вашем устройстве.
2. Откройте приложение и выполните поиск доступных устройств.
3. Подключитесь к выбранному устройству.
4. Начните обмен сообщениями.

## Технологии

- **Android SDK**
- **Wi-Fi Direct**
- **mDNS (Multicast DNS)**


## Преимущества

- **Работа без интернета**: Идеально для использования в условиях отсутствия подключения к интернету.
- **Энергоэффективность**: Wi-Fi Direct более энергоэффективен по сравнению с Wi-Fi Hotspot.
- **Высокая скорость передачи данных**: Быстрее, чем Bluetooth, что делает его идеальным для обмена сообщениями.
